📖 O Que É Isso?

O OiaATabuada é um aplicativo inteligente e seguro que ajuda crianças a praticarem matemática enquanto garantem um uso consciente do dispositivo móvel! 📱➗

-----------------------------------------------------------------------------------------------
🎯 Como Funciona?

    A criança precisa resolver 3 problemas de matemática corretamente para ganhar tempo de uso

    Após completar o quiz, o app libera o dispositivo por 10 minutos ⏰

    Quando o tempo acaba... adivinha? Mais matemática! 🔁

    Totalmente protegido contra saídas, desinstalações e travamentos! 🛡️
-----------------------------------------------------------------------------------------------
🧩 Funcionalidades Principais
-----------------------------------------------------------------------------------------------
🧠 Quiz Matemático Inteligente

    ➕ Adição

    ➖ Subtração

    ✖️ Multiplicação

    ➗ Divisão

    🔀 Operações mistas

    📊 Dificuldade progressiva
-----------------------------------------------------------------------------------------------
👨‍👩‍👧‍👦 Controle Parental

    🔐 Senha dos pais (acesso com botão volume -)

    📊 Histórico completo de atividades

    📤 Exportar relatórios (WhatsApp, Email, etc.)

    🔄 Alterar senha de acesso
    
-----------------------------------------------------------------------------------------------
🛡️ Proteção Total

    🚫 Bloqueio de botões (voltar, home, recentes)

    🔒 Impede desinstalação

    📵 Modo quiosque (só sai com senha)

    👀 Monitoramento em tempo integral
    
-----------------------------------------------------------------------------------------------
🎮 Como Usar?
-----------------------------------------------------------------------------------------------
Para a Criança 👶

    📱 Pega o celular

    🧮 Resolve 3 problemas de matemática

    🎉 Ganha 10 minutos de uso livre!

    ⏰ Quando o tempo acaba... repete tudo!
    
-----------------------------------------------------------------------------------------------
Para os Pais 👨‍👩

    🔉 Pressione Volume - para acessar o menu pais

    🔐 Digite a senha (padrão: 123456)

    📊 Veja o histórico de atividades

    📤 Exporte relatórios para acompanhar o progresso

    🔄 Altere a senha se necessário

❓ Perguntas Frequentes:

❔ O app é gratuito?
✅ Sim! Totalmente gratuito e open-source!

❌ Não! Funciona 100% offline.
